Teória elemental de conjuntos

¿Qué es un conjunto ?

Se le llama conjunto a una colección o una agrupación de elementos perfectamente bien definidos y diferenciados dentro de un todo.

Requisitos para formar un conjunto

Conjunto vacío

El conjunto vacío es el que carece de elementos. Se denota por Ø

Subconjunto

Si cada elemento de un conjunto A es también elemento de un conjunto B, entonces se dice que A es un subconjunto de B. Se dice también que A está contenido en B o que B contiene a A. La relación de subconjunto viene dada por: A ⊂ B o B ⊂ A

Ejemplo:

Observe que todos los elementos del conjunto C están en el conjunto A; por tanto C ⊂ A. Asimismo, podemos notar que C ⊂ B. Sin embargo, no todos los elementos de B están A, por lo que podemos denotar que B ⊄ A. Además, A ⊄ B, A ⊄ C y B ⊄ C.

Pertenencia

Cuando se relaciona a un elemento con el conjunto al cual pertenece se utiliza el símbolo ∊ que se emplea para abreviar pertenece o es miembro de o es elemento de. Cuando un elemento a pertenece al conjunto A se denota por:

a ∈ A (“a pertenece a A”)

Para expresar que el elemento a no pertenece al conjunto A se representa con:

a ∉ A (“a no pertenece a A”)
